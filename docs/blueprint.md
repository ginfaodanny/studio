# **App Name**: Profile Canvas

## Core Features:

- Profile Information Display: Display personal information, including name, bio, and profile picture.
- Project Showcase: Display a portfolio of projects with descriptions and links to GitHub repositories.
- Skills Visualization: Showcase technical skills and proficiencies using interactive charts.
- Contact Form: Enable visitors to send direct messages through a simple contact form.
- Theme Customization: Allow users to switch between light and dark themes for optimal viewing experience.
- AI-Powered Content Refinement: Suggest improvements to the user's profile and project descriptions using a generative AI tool. This helps in enhancing clarity and impact.

## Style Guidelines:

- Primary color: A calming, professional blue (#4A77D4) to inspire confidence and reliability.
- Background color: Light, desaturated off-white (#F0F2F5) for a clean and modern look.
- Accent color: A subtle, analogous purple (#704AD4) for highlights and interactive elements.
- Font: 'Inter', a sans-serif font known for its readability and modern aesthetic, will be used for both headings and body text.
- Simple, professional icons to represent skills and project categories.
- A clean, single-page layout with clear sections for easy navigation.
- Subtle transitions and hover effects to provide visual feedback and enhance user engagement.
