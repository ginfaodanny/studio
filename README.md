# Profile Canvas

This is a Next.js portfolio website bootstrapped with Firebase Studio.

## Overview

Profile Canvas is a modern, single-page portfolio designed to showcase your skills and projects in a visually appealing way. It features a clean design, theme customization (light/dark mode), and an innovative AI-powered content refinement tool to help you craft the perfect descriptions for your profile and projects.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:9002](http://localhost:9002) with your browser to see the result.

You can start editing the page by modifying `src/app/page.tsx`. The page auto-updates as you edit the file.

## Features

- **Profile Display**: Showcase your personal information, including your name, bio, and profile picture.
- **Project Showcase**: Display your work with descriptions and links to repositories.
- **Skills Visualization**: Use interactive charts to highlight your technical abilities.
- **Contact Form**: Allow visitors to get in touch.
- **Theme Customization**: Switch between light and dark modes.
- **AI Content Refinement**: Leverage generative AI to improve your written content.
